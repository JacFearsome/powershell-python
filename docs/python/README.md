### Conditional Statements
```python

```
```python
if 2 > 1:

```
```python
if 1 > 2:
  print("1 is greater than 2")
else:
  print("1 is not greater than 2")
```
```python
if 1 > 2:
  print("1 is greater than 2")
elif 2 > 1:
  print("1 is not greater than 2")
else:
  print("1 is equal to 2")
```
